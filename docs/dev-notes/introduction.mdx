# Introduction

## Running the Coral V5+

Initializing the repository:

```sh
sh initialize.sh
```

Run these commands to start Coral V5+ in watch mode:

In the `client/` folder in one terminal:
```sh
npm run watch
```

In the `server/` folder in a separate terminal:
```sh
npm run watch
```
